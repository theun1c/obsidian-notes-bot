# obsidian-notes-bot

чета для себя и не факт что не забью

### Флоу работы:

- Пишем в бота какой-то щитпост ->
- он сохраняет заметку в формате .md в другой [репо](https://github.com/theun1c/obsidian-notes-inbox) (типо inbox) (репо отвечает за Unsorted папку в obsi-vault) ->
- на своем компе я линкую репозиторий со своим obsidian-vault и делаю pull ->
- после переноса заметок они становятся deleted в git и я делаю push ->
- заметки пропадают внутри репозитория ->
- в боте я делаю также git pull и заметки из-за того что их уже нет в репо удаляются

> Получается так что в таком флоу я не засоряю сервак и собственный репозиторий, который является посредником (inbox)

### Фичи:

- получение заметки через бота
- создание по тексту заметки .md файла
- шаблонизация
- работа с гитом

> должен получиться почти что автоматизированный процесс получения и сохранения заметок через тг в obsi

### Что пока не продумано:

- самостоятельный пулл уже на локальной машине
- самостоятельный push на репо чтобы сделать удаление файлов

> но в целом на это все реально написать скрипты
